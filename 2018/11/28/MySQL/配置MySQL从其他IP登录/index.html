配置MySQL从其他IP登录
2018-11-28|MySQL|
Word count: 260|Reading time: 1 min

默认情况下，MySQL服务器只能从本机登录，无法从其他电脑登录的，需要进行配置。MySQL只能设置一个IP，就是服务器的IP。

找到最常用的那个配置文件mysqld.cnf，对于比较新的MySQL，文件在/etc/mysql/mysql.conf.d/mysqld.cnf。找到#bind-address = 127.0.0.1这行，注释掉然后重启MySQL。

现在设置了MySQL允许从其他IP登录，再进行不同用户的IP登录设置。

其实目的就是让数据库mysql里的user表里的host项，从localhost改称”%”或指定的IP。注意同样的用户名操作一次就行了，多了会重复，虽然没问题但是不严谨。

登录的命令是: mysql -u用户 -p密码 -hAddress，Address是MySQL服务器的地址，3304是默认的端口，一般不必修改
