Mysql批量插入随机数据
2019-08-27|MySQL|
Word count: 341|Reading time: 1 min

有时为了测试数据库的操作，需要有一些假的随机数据，所以研究一下插入随机数据的方法

产生随机数

SELECT RAND() * 10000; # 产生0到1000间的随机数

SELECT FLOOR(RAND() * 10000) # 对应产生相应的整数

SELECT MD5(RAND() * 10000) # 使用md5()产生32位随机字符串

随机产生时间值
