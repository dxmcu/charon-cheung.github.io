MySQL常用命令和Workbench常用配置
2018-07-12|MySQL|
Word count: 545|Reading time: 2 min
命令功能
net start mysql启动(windows)
net stop mysql停止(windows)
service mysqld start启动(Linux)
service mysqld stop停止(Linux)
service mysqld restart重启(Linux)
service mysqld status状态(Linux)
mysql -uroot -p123456进入MySQL
show engines查看数据库引擎
SELECT user,host from mysql.usr查看所有用户
mysql -v(Linux)查看MySQL版本
select version()查看MySQL版本
show variables like ‘port’查看MySQL端口(默认3304)
命令末尾+\G;查询结果竖直排列
常用备份方法
mysqldump -uroot -p123456 ipacs5000 >bak.sql备份数据库ipacs5000
mysqldump -uroot -p123456 –databases test ipacs5000 >bak.sql备份库test和ipacs5000
mysqldump -uroot -p123456 -all-databases > bak.sql备份所有库
mysqldump -uroot -p -d test blog >E:\bak.sql备份test库的blog表
SELECT *FROM eventoperate INTO OUTFILE ‘D:/t.csv’备份eventoperate表
mysql -uroot -p123456 ipacs5000 (–default-character-set=utf8) <bak.sql还原数据库
SELECT COUNT(*) FROM table获得表行数
DESC 表名查看表的字段信息
SHOW CREATE TABLE 表名查看表的所有信息
SHOW PROCESSLIST显示所有活动线程
show variables like ‘slow%’;慢查询
show global status like ‘%connect%’;显示连接数
TRUNCATE table_name清空表
SHOW CREATE TABLE TEST查询建表语句

某一列放到第一位置也要用命令执行，关键字FIRST

对表的列或索引进行操作后，最好执行一下truncate命令，否则再执行其他命令容易出错

将A表的数据插入到B表中

如果某个字段有重复的，而插入数据的时候要不重复，可以加上DISTINCT或者GROUP BY
