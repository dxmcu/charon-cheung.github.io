set和map
2020-03-15|C++C++ 模板与STL|
Word count: 389|Reading time: 1 min

set

set的元素有序不重复，而且能根据元素的值自动进行排序。set中的键值不能直接修改，只能先删除再插入。底层采用红黑树。

set不支持随机访问，只能使用迭代器去访问。由于set放入一个元素就会调整这个元素的位置，把它放到合适的位置，所以set中只有一个insert插入操作。

multiset底层也是红黑树，但允许有重复数据

map

map适合存储一个数据字典，并要求方便地根据key找value。

Map节点有一个Key和Value两个元素,Key不重复,Value可以重复。map可以通过key改变value的值

底层也是红黑树，插入删除查找的时间复杂度为O(logN)

map支持at函数和[]下标访问，这是set没有的
