Cgicc的安装配置见Qt ROS CGI三者的联合使用

我们写一个程序，最终编译的可执行文件是test，放到mini-httpd的网络目录里，执行的时候在网页里访问 http://192.168.1.5:8080/test?name=abc，我们希望在程序里获知name的值是abc

类cgicc::Cgicc实例化后，会处理HTTP服务器传给CGI脚本的数据，所有错误是用exceptions处理的所以最好用try catch处理
try {
cgicc::Cgicc cgi;
}

catch(exception& e) {
// Caught a standard library exception
}
