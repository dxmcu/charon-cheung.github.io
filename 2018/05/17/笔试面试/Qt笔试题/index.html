Qt笔试问题
2018-05-17|面试笔试|
Word count: 1.9k|Reading time: 6 min

1. 以下描述正确的是：
a.元对象系统可以对信号使用模板
b.元对象系统可以对槽使用模板
c.Qt可在多个平台的应用程序中完全应用模板
d.Qt不能在多个平台的应用程序中完全应用模板

2. 判断：函数指针是否可以作为信号和槽的参数？

解答：不能。这种情况下，继承是一个不错的替代方法。

3. 不是元对象代码实现的特征的是：
a.提供信号/槽机制
b.Class.forName( )在运行的时候返回类名称
c.tr()用于国际化中的字符串翻译
d.metaObject( )返回该类所关联的元对象

解答：一般说，元对象系统包括QObject类、Q_OBJECT宏、元对象编译器。也可以说包括：信号与槽、QObject::metaObject绑定元对象、QMetaObject::className提供类名称、tr()和trUtf8()用于国际化、QObject::inherits继承信息、属性系统、qobject_cast。b的表述不正确。

4. 以下关于元对象编译器的叙述不正确的是：
a.元对象编译器不展开#define，但可展开#include
b.元对象编译器不展开#include
c.忽略遇到的所有预处理程序
d.无法处理所有C++语法

解答：moc不扩充宏，不能使用一个宏来声明一个信号/槽，也不能使用宏为一个QObject定义一个基类.元对象编译器并不展开#include或者#define，它忽略所遇到的所有预处理指示。moc 并不能处理所有的C++ ，比如类模板不能包含信号和槽。由此可见a是不正确的。

5. 有如下声明：

关于属性声明的位置以下正确的是：
a.原位置正确 ① ② ③ ④
b.原位置不正确，应为 ① ③ ④ ②
c.原位置不正确，应为 ① ③ ② ④

解答：属性的声明应该放在含有相应的读写函数的公有部分之前、Q_OBJECT宏之后。这里应该选b

6. 以下关于属性的叙述不正确的是：
a.基于元对象系统
b.在类声明中用宏声明
c.属性不是一个类的成员
d.属性只能在继承于QObject的子类中声明

解答：属性属于元对象系统的一部分，使用Q_PROPERTY声明，只能用于QObject及子类。所以c叙述不正确。

7.以下关于QObject类叙述不正确的是：
a.是所有Qt对象的基类
b.QObject对象树是一个静态的QObjectList类对象object_trees
c.object_trees链表中存有部分对象指针
d.通过object_trees链表可查询到所有的对象

解答： a错误，QEvent,QGraphicsItem, QRunnable就不是QObject的子类，它只属于元对象系统。

8. 以下关于QObject类定时器的叙述正确的是：
a.当处理器正忙于其它事件的处理时，定时器仍可以立即触发
b.当定时器触发时，应用程序会发送QTimerEvent
c.用startTimer( )开始一个定时器，如果成功则返回0
d.用startTimer( )开始一个定时器，如果失败则返回-1

解答：在事件循环中，处理器按照事件队列的顺序来处理定时器事件。当处理器正忙于其它事件处理时，定时器就不能立即处理。定时器触发后会调用registerTimer，然后或者发送QZeroTimerEvent或QTimerEvent；startTimer成功会返回定时器的ID，失败则返回0。所以选b

9. 以下是QObject提供的字符串翻译函数的选项是：
a.translate( ) , trUtf8 ( )
b.translation( )
c.tr ( ) , trUtf_8 ( )
d.tr ( ) , trUtf8 ( )

解答：translate()用于2D图元的平移变换，translation()用于Qt3D模块的平移变换，应当选d

10.以下关于元对象的叙述不正确的是：
a.每个在类声明中含有Q_OBJECT宏的类都有一个对应的元对象
b.每个在类声明中含有Q_OBJECT宏的类都有一个对应的QMetaObject类对象
c.元对象是动态的
d.一个类可能含有多个元对象与之对应

11. 以下关于Qt风格叙述正确的是：
a.Qt风格类的基类是QStyle
b.QStyle的多数成员函数既有声明也有实现
c.只能在QCommonStyle类中通过重载来实现

解答：QStyle是所有风格类的基类，它控制着所有的部件的界面风格，它定义了大量的枚举类型和十几个函数。但大多数函数只是声明而没有实现，只实现了3个函数drawItem(), itemRect(), visualRect()。现在QStyle只有一个子类QCommonStyle,一个孙类QProxyStyle。可以重载这三个类实现自己的风格。应当选a
参考：Qt外观之QStyle

12. 判断：布局部件只有QHBoxLayout, QVBoxLayout, QGridLayout。

解答：错误。QLayout继承自QObject和QLayoutItem。QLayout的子类有QBoxLayout, QGridLayout, QFormLayout, QStackedLayout。

a.仅拷贝指针而不拷贝条目本身的，称“浅拷贝”
b.不允许只拷贝指针而不拷贝条目本身
