字符串条件判断

注意：

  • 左右方括号都要留空位，if后面也要有空位
  • Shell下使用等号赋值时，左右两边不能有空位
  • 一个等号和两个等号都可以
  • then要另起一行，以if开头，以fi结尾

统计文件行数、字节、字数

wc命令，选项-l, -c, -w分别统计行数、字节、字数，可统计多文件，但不能统计目录。

统计目录中的文件个数（包括文件夹0

ls | wc -l


当前文件夹或文件大小

当前文件夹：du -smh
当前文件夹下所有文件和子文件夹： du -mh .
某文件: du -smh file

当前目录下大于400M的文件夹和文件，并从小到大排序

判断文件/文件夹是否存在

-d判断后面的文件夹是否存在
-f判断是否存在某文件
-s判断是否存在某文件，且大小不为0，这个比-f更有用
